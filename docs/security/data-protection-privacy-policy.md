# Data Protection and Privacy Policy

## Purpose

This policy ensures compliance with ISO/IEC 27018 and data protection regulations for personally identifiable information (PII) processed in cloud environments.

## Scope

Applies to all PII collected, processed, or stored by the Chiral platform.

## Key Principles

### 1. Lawfulness, Fairness, and Transparency
- Process PII lawfully and transparently
- Provide clear privacy notices
- Obtain consent where required

### 2. Purpose Limitation
- Collect PII only for specified purposes
- Not use PII for incompatible purposes
- Document all processing purposes

### 3. Data Minimization
- Collect only necessary PII
- Retain PII only as long as needed
- Regularly review and delete unnecessary data

### 4. Accuracy
- Keep PII accurate and up to date
- Implement data validation processes
- Allow data subjects to correct inaccuracies

### 5. Storage Limitation
- Establish retention schedules
- Implement automated data deletion
- Document retention justifications

### 6. Integrity and Confidentiality
- Protect PII against unauthorized access
- Use encryption for data at rest and in transit
- Implement access controls

### 7. Accountability
- Maintain records of processing activities
- Conduct privacy impact assessments
- Implement privacy by design

## Cloud-Specific Controls

### ISO 27018 Implementation
- Implement shared responsibility model
- Ensure CSP compliance with privacy standards
- Maintain audit trails for PII processing
- Provide data subject rights mechanisms

## Data Subject Rights

- Right to access personal data
- Right to rectification
- Right to erasure
- Right to data portability
- Right to object to processing

## Breach Notification

- Report breaches within 72 hours
- Notify affected data subjects
- Cooperate with supervisory authorities

## Review and Update

This policy is reviewed annually and updated as needed.
